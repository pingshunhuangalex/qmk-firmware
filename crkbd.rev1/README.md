Designed by [Foostan], the [Corne] is a 40% (`3 Ã— 6`) split keyboard with column staggered keys and 3 thumb keys, based on [Helix]. `CRKBD` stands for Corne Keyboard. It is a compact keyboard featuring per-key LED, backlight / underglow, hotswap compatibility, and OLED support.

## Build via QMK

```console
qmk compile -kb crkbd/rev1 -km keydo
```
